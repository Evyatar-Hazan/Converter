# Universal Data Converter

A unified, privacy-first web app for converting between JSON, CSV, and XML formats. All tools run 100% in your browser—no uploads, no tracking.

---

## 🚀 Features
- **JSON ⇄ CSV ⇄ XML**: Four conversion tools in אחד אתר
- **Privacy First**: כל העיבוד בדפדפן שלך בלבד
- **UI מודרני**: רספונסיבי, נגיש, מהיר
- **הורדה בלחיצה**: שמירה מיידית של התוצאה
- **localStorage**: שומר קלט אוטומטית
- **קוד פתוח**: HTML5, CSS3, JavaScript (ES6+)

